# Time Series Demand Forecasting System

## Overview

This project predicts future product sales based on historical demand and external factors using ARIMA, Prophet, and Random Forest models.

## Structure

- `data/`: Raw and processed data
- `models/`: Trained models
- `src/`: Source code for preprocessing, training, and forecasting
- `app/`: Streamlit web application
- `notebooks/`: EDA and experiments

## Setup

1. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

2. Run the web app:

   ```bash
   streamlit run app/app.py
   ```
